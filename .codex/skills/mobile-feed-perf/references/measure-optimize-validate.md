# Measure -> Optimize -> Validate

## Measure
- Capture baseline FPS/scroll smoothness/startup timings.
- Record affected screens and reproducible scenario.

## Optimize
- Prefer localized state updates.
- Avoid unnecessary re-renders in card trees.
- Keep virtualization settings explicit and justified.

## Validate
- Re-run same scenario and compare to baseline.
- Report metric deltas and risk tradeoffs.
